MailSystem is mailsystem.org.ua integration software with Wordpress

== Description ==

Integration mailsystem.org.ua with Wordpress.


== Installation ==

1. Place the 'mailsystem' folder in your '/wp-content/plugins/' directory.
2. Activate mailsystem.
3. Visit 'Mailsystem > Settings' and adjust your configuration.
5. Use Plugin
